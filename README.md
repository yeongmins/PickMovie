# PickMovie

## 1) PickMovie 개요
PickMovie는 사용자가 보고 싶은 **영화/TV 콘텐츠를 탐색**하고, 작품의 **상세 정보(포스터/평점/개봉·방영일 등)**를 확인하며, 마음에 드는 작품을 **찜(즐겨찾기)** 으로 저장해 관리할 수 있는 콘텐츠 탐색 서비스입니다.  
TMDB 기반 데이터를 활용해 메인 화면에서 다양한 콘텐츠를 보여주고, 상세 페이지에서 작품 정보와 시청 가능 OTT(프로바이더) 정보를 제공합니다.

---

## 2) 핵심 기능
- **콘텐츠 탐색**
  - 메인 화면에서 영화/TV 콘텐츠 리스트(캐러셀/그리드 등) 노출
  - 포스터, 제목, 평점, 개봉/방영일 등 기본 정보 표시

- **콘텐츠 상세 페이지**
  - 작품 기본 정보 표시 (포스터/배경, 제목, 평점, 개봉·방영일 등)
  - 트레일러(영상) 표시
  - 시청 가능 OTT(프로바이더) 목록 표시

- **찜(즐겨찾기)**
  - 작품을 찜/해제하여 즐겨찾기 목록으로 관리
  - 로그인 상태에 따른 찜 동작(가능한 경우 프로젝트 정책에 맞게)

- **시리즈/시즌 표시(TV)**
  - TV 작품의 시즌 정보를 목록 형태로 표시
  - 시즌 항목을 콘텐츠 카드 형태로 보여주고 상세 페이지로 이동

  - **검색(Picky)**
  - 자연어로 “요즘 볼 만한 감성 영화” 같은 요청을 빠르게 찾을 수 있도록 보조
